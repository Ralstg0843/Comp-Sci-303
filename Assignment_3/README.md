This project has 3 parts:

1. A template Queue class with push, pop, front, size, empty, and move_to_rear.
2. A recursive linear search that finds the last time a value appears in a vector.
3. Insertion sort that works on a linked list.


To compile everything:g++ Queue.cpp recursive_search.cpp insertion_sort_list.cpp -o program
./program
To test the queue: g++ Queue.cpp -o queueTest
./queueTest
To test the insertion sort: g++ insertion_sort_list.cpp -o sortTest
./sortTest
